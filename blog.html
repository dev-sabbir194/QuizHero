1.What is the difference between Map, Filter, ForEach and Find

map and filter create new arrays, while forEach and find do not. map transforms each element in the array, filter returns a subset of the original array, forEach executes a function on each element, and find returns the first element that satisfies a testing function.

2.What is the difference between localstorage, sectionstorage and cookies

localStorage and sessionStorage store data in the user's browser and are accessible only on the client side, whereas cookies are stored in the user's browser and can be accessed by both the server and the client. localStorage data is persistent, sessionStorage data is available only for the duration of the session, and cookies can have an expiration date set.

3.What is Eventloop? And how does it work?

The event loop is a mechanism in JavaScript that allows for handling asynchronous events and executing code in a non-blocking way by executing tasks in a queue one by one, ensuring that the user interface remains responsive and other tasks can continue to execute while waiting for an asynchronous operation to complete.
